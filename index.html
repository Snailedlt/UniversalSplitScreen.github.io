Universal Split Screen

Split screen multiplayer for any PC game with multiple keyboards, mice and controllers

Quick start

What is Universal Split Screen?

Universal Split Screen is a standalone application that directs Windows input to enable local multiplayer in PC games with keyboards, mice and controllers. Play local co-op with one game copy.

Easy to install

Download and run without administrator privileges. Full guides and settings presets are available for all tested games.

Wide game support

Universal Split Screen can work with any game designed for keyboard and mouse. All major input techniques are supported in the options, meaning there are many compatible games.

100% Open Source

100% of the code is open source on GitHub under the MIT license. Submit patches or pull requests and use it freely in your projects.
